# MultiCam (Windows package baseline)

Репозиторий для совместной работы над переносом `MultiCam` на Linux/Jetson.

## Что внутри

- `MultiCam.exe` и `nncam.dll` — исходная Windows-сборка для камеры.
- `Manual.pdf` — руководство пользователя.
- `calibration05.txt`, `calibration10.txt`, `calibration15.txt` — калибровочные матрицы.
- `Reference/settings.txt` — пример параметров съёмки.

## Что исключено из git

Крупные бинарные зависимости и отладочные артефакты (OpenCV bundle, PDB и т. п.) исключены через `.gitignore`.

## Ближайшая цель

Собрать Linux-пайплайн для:

1. получения кадров с камеры через `libnncam.so`;
2. разрезки/регистрации 16 каналов;
3. расчёта вегетационных индексов;
4. интеграции с геотегами и моделью анализа.
